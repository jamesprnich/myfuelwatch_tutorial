# Python .format

You can use the `str.format()` method to do variable substitutions and value formatting. 

A  simple example:

{% code-tabs %}
{% code-tabs-item title="python\_format\_example.py" %}
```python
print("Sam has {} pets.".format(2))
```
{% endcode-tabs-item %}
{% endcode-tabs %}

{% code-tabs %}
{% code-tabs-item title="Output" %}
```text
Sam has 2 pets.
```
{% endcode-tabs-item %}
{% endcode-tabs %}
